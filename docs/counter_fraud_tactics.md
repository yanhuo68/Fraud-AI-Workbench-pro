# Counter-Fraud Tactics & Fraudster Behaviors

This document describes common tactics used by fraudsters and how models detect them.

---

# 1. Account Takeover (ATO)
- Sudden activity change
- Large transfers
- Zero-balance manipulation

---

# 2. Mule Account Networks
- Multiple senders → one mule
- Mule → cash-out
- Inconsistent balances

---

# 3. Smurfing (Structuring)
- Small repeated deposits
- One large withdrawal

---

# 4. Rapid Multi-Hop Laundering
- Sequential transfers across accounts
- Occurs within same hour

---

# 5. Synthetic Identity Fraud
- Brand new accounts
- No activity for 100+ hours
- Sudden large transactions

---

# 6. Social Engineering + Cash-Out
- Fraudster steals credentials
- User account drained quickly

---

# Purpose
This helps LLM agents recognize evolving fraud patterns.
