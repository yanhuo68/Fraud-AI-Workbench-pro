# Streamlit Dashboard Metrics Guide

This document defines which metrics the dashboard must display for each model.

---

# 1. Isolation Forest Metrics
- Anomaly score histogram
- Score distribution for fraud vs non-fraud
- Precision@k
- Anomaly ranking table

---

# 2. Random Forest Metrics
- Precision / Recall / F1
- AUC
- Confusion Matrix
- Probability distribution
- Feature importances chart

---

# 3. LOF Metrics
- Outlier factor distribution
- Top anomaly transactions
- Fraud overlap statistics

---

# 4. Dataset Summary Metrics
- Fraud rate
- Amount distribution
- Transaction type breakdown
- Missing value counts

---

# Purpose
Provides consistent metric structure across all ML algorithms.
