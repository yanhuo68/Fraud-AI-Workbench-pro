# ML Algorithms for Fraud Detection

Recommended algorithms:

1. **Isolation Forest** — anomaly detection  
2. **LOF (Local Outlier Factor)** — local density anomalies  
3. **One-Class SVM** — boundary-based anomaly detection  
4. **Random Forest Classifier** — top supervised baseline  
5. **Gradient Boosting** — strong tabular model  
6. **XGBoost** — advanced fraud classifier  
7. **Logistic Regression** — interpretable baseline  
8. **Neural Networks (MLP)** — moderate performance  
9. **LSTM (future)** — sequential fraud patterns
