# Banking Fraud Detection Metrics

Applies to:
- Card fraud
- Account takeover (ATO)
- Wire fraud
- AML-related suspicious activity

---

## 1. Transaction-Level Metrics

### Authorization Fraud Rate
Fraudulent Transactions / Total Authorized Transactions

### Chargeback Rate
Chargebacks / Total Transactions

### Fraud Dollars per $1000 Processed
Fraud Loss / Total Volume * 1000

---

## 2. Account Takeover Metrics

### ATO Detection Rate
Detected ATO Cases / Total ATO Cases

### Time to Detect (TTD)
Average time between compromise and detection.

### Compromised Account Ratio
Compromised Accounts / Active Accounts

---

## 3. AML Monitoring Metrics

### Suspicious Activity Report (SAR) Conversion Rate
SAR Filed / Alerts Generated

### Alert-to-Case Conversion
Cases Created / Alerts Generated

### Alert Backlog Size
Pending Alerts

---

## 4. Customer Impact Metrics

### Customer Friction Rate
Legitimate Transactions Blocked / Total Legitimate Transactions
