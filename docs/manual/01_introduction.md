# 01. Introduction & Sentinel Operator

The **Home Page** is your central command center for the Sentinel AI Workbench. It provides a visual gateway to all intelligence modules and features an interactive AI assistant to help you navigate the platform.

## 🚀 Home Page Features

- **Intelligence Grid**: A visual overview of all available modules (Data Hub, SQL Assistant, Graph Navigator, etc.). Each card provides a direct link to the respective page and a demo overview.
- **Sentinel Explorer**: For unauthenticated users, the Home page promotes login/registration to unlock the full investigative suite.
- **Dynamic Debug Console**: At the bottom of the page, you can see your current Role and Permission count, helping you understand your access level.

## 💬 Sentinel Operator

The **Sentinel Operator** is an AI-powered guide designed to assist with platform navigation and investigative methodology.

### How to Use:
1. Locate the **Sentinel Operator** chat box on the right side of the Home page.
2. Ask any question regarding the platform, such as:
   - *"How do I upload a new CSV?"*
   - *"What is a Fraud Ring in Graph RAG?"*
   - *"How can I link an IP to a user account?"*
3. The Operator maintains a short conversation history (up to 4 turns) to provide context-aware assistance.
4. Use the **Purge Chat History** button to reset the assistant.

> [!TIP]
> If you are lost or unsure which tool to use for a specific fraud investigation, ask the Sentinel Operator for a recommended workflow.
